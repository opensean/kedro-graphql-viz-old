A component for a PipelineTemplate entity, supports serialization and de-serialization of PipelineTemplate objects and can be used from
both backend and frontend.

```ts
const pipelineTemplate = PipelineTemplate.from({
  id: "100000000000000000000001",
  name: "example00",
  version: "0.1.0",
  describe: "#### Pipeline execution order ####\nInputs: parameters, params:example, text_in\n\necho_node\n\nOutputs: text_out\n##################################",
  nodes: mockNodes,
  parameters: mockParameters,
  inputs: [
    "text_in",
    "params:example",
    "parameters"
  ],
  outputs: [
    "text_out"
  ]
});
```
